# single-elimination
Single elimination brackets

## Локальный запуск

1. Установите зависимости:
   ```sh
   npm install
   ```
2. Запустите приложение в режиме разработки:
   ```sh
   npm run dev
   ```
   Приложение будет доступно по адресу http://localhost:5173 (или указанному в терминале).

## Сборка для продакшена

```sh
npm run build
```

## Предпросмотр собранного приложения

```sh
npm run preview
```

## Запуск тестов

```sh
npm test
```

Тесты запускаются с помощью Jest. Все тесты должны проходить успешно.

## Основные страницы

- **Главная** (`/`) — история турниров, запуск и удаление.
- **Настройка турнира** (`/setup`) — импорт участников (CSV/ручной ввод), настройка параметров и запуск турнира.
- **Турнир** (`/play/{id}`) — поэтапное проведение матчей и выбор победителя.
